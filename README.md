# remove-mac-made-files
This removes .DS_STORE files and other ._files.xxx from the location specified.  This is especially useful for placing my music collection on an SD card to be read into my car.

## How to use
**Prerequisites:**
 - only for use on MacOS (possibly Linux)
 - volume must be accessible

1. open remove-mac-made-files.sh
2. enter the location of the volume from your current directory
3. wait until the line "Mac-made files have been removed from $VOLUME_PATH." is displayed and the state has been saved
